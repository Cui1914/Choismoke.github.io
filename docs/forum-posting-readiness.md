# 烟灰缸发帖与评论准备说明

这份文档只讲“真实发帖”和“真实评论”这一层当前准备到了哪里。

## 当前已经补上的部分

- 发帖接口入口
- 评论接口入口
- 发帖页对应的提交位置
- 帖子详情页对应的评论提交位置

## 现在已经有的接口入口

- `/api/forum/threads`
- `/api/forum/comments`

## 这代表什么

你现在看到的发帖页和帖子详情页，已经不是单纯的展示页了。

它们已经有了后面接真实内容系统时应该放数据的位置。

## 后面接上真实系统后，会发生什么

### 1. 帖子会真的发布

- 发帖后会进入论坛首页
- 会进入对应分类页
- 会进入用户个人主页的最近发帖

### 2. 评论会真的保存

- 用户在帖子页输入回复后，会真实保存
- 回复数量会变化
- 互动通知也能跟着接起来

## 你现在只需要知道的结论

我已经把“发帖系统”和“评论系统”的接口位置和页面结构铺好了。

后面如果继续推进，就不再是画发帖页和评论区，而是开始接真实内容数据。
