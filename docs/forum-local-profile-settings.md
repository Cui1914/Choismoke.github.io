# 烟灰缸本地账号资料说明

这份文档只讲“用户资料和设置”这一层，现在已经有哪些内容会真正写入本地。

## 当前已经支持本地写入的内容

- 注册账号
- 用户名
- 个人简介
- 关注列表可见性
- 粉丝列表可见性
- 私信规则
- GitHub 链接
- Bilibili 链接
- 公开邮箱

## 当前相关接口

- `/api/auth/register`
- `/api/auth/session`
- `/api/auth/settings`

## 当前本地数据文件

- `community-app/data/auth-db.json`

## 这代表什么

现在账号系统不再只是“注册预览”和“设置预览”。

至少在第一版里：

- 注册会写入本地账号数据
- 设置项已经有真实保存位置
- 设置页已经开始从本地账号数据读取

## 你现在只需要知道的结论

烟灰缸现在已经不只是“论坛页面像真的”，而是连账号资料和设置都开始具备本地真实保存的能力了。
